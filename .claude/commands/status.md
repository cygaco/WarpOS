Show the current project status:

1. **Branch info** — current branch, commits ahead/behind master, last commit message + date
2. **Recent commits** — last 10 commits with one-line messages
3. **Uncommitted changes** — staged, unstaged, untracked files
4. **Build health** — quick `npm run build` pass/fail
5. **Open work** — check for TODO/FIXME/HACK comments in src/

Report as a concise dashboard. Keep it brief — this is a status check, not an audit.
