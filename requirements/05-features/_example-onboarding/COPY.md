|---------|------|
| **Heading** | Import Your Data |
| **Subheading** | Upload a file or paste text — we'll extract the important stuff. |

| **Paste button** | Paste Text |

| **File rejected** | That file type isn't supported. Try PDF, DOCX, TXT, or MD. |
